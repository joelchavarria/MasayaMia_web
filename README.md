# Masaya Mía Web

Sitio institucional liviano para Masaya Mía Café en Masaya, Nicaragua.

## Desarrollo

```bash
npm install
npm run dev
```

## Producción

```bash
npm run build
npm run preview
```

## SEO y analítica

El sitio incluye metadatos sociales, sitemap, robots.txt y datos estructurados
para negocio local. Para activar Google Analytics en Vercel, configurar la
variable:

```bash
VITE_GA_MEASUREMENT_ID=G-XXXXXXXXXX
```
